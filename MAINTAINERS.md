# Maintainers

This file lists the people with merge authority on `nist-agent-passport`.
It is the authoritative record — if a name isn't here, that person doesn't
have decision-making authority over the project.

## What maintainers do

- Review and merge pull requests (see [CONTRIBUTING.md](./CONTRIBUTING.md)
  for the review criteria)
- Triage incoming issues
- Tag releases following [Semantic Versioning](https://semver.org/spec/v2.0.0.html)
- Coordinate security disclosures per [SECURITY.md](./SECURITY.md)
- Maintain alignment with the [NIST AI Agent Standards
  Initiative](https://www.nist.gov/artificial-intelligence/ai-agent-standards-initiative)
  and the standards baseline documented in `CLAUDE.md`

## Becoming a maintainer

The project is small. Maintainership is extended on the basis of sustained
contribution — typically several merged PRs plus a demonstrated grasp of
the standards baseline (NIST 800-63-3, RFC 8693, OIDC + PKCE). Open an
issue if you'd like to discuss it.
